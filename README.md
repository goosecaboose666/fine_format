## Features

*   **Intelligent Content Processing**: Support for multiple file formats and URLs.
*   **Automated Dataset Engineering**: Q&A generation using state-of-the-art LLMs.
*   **Quality Assurance**: Automated validation and scoring of generated data.
*   **Dataset Optimization**: Export ready-to-use JSONL files for fine-tuning.

## Quick Start

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Run locally with Netlify CLI: `netlify dev`.

For detailed instructions, please refer to the [Deployment Guide](docs/deployment/DEPLOYMENT_GUIDE.md).
